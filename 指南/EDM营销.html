EDM营销工具 ​

EDM营销，全称为Email Direct Marketing（电子邮件直复营销），是一种通过电子邮件与目标客户进行沟通，向其直接传达营销信息，用以促进销售的营销手段。

邮箱工具：Maddy-mail ​

第一步 下载应用 ​

1Panel 面板应用商店搜索 Maddy-mail 应用，下载安装。

安装时记得一定要打开：端口外部访问。

第二步 二级域名准备 ​

在域名解析处添加解析，推荐mail.example.com

主机记录值记录类型记录值
mailA填写自己的云服务地址

第三步 准备证书 ​

按顺序找，1panel 操作页面的左侧栏，从上往下第三个，点开后扩展>证书，进去后配置好acme账户与dns账户。

acme账户

DNS 账户

  • 阿里云AccessKey
  • 腾讯云AccessKey

注意

如果购买或申请了通用型SSL证书，此处并不能直接使用一级域名的通用型证书，仍然需要上述操作。

以上准备工作完成后，点申请证书，主域名填选择议mail.example.com

勾选 自动续签 和 推送到本地，目录为 /var/lib/docker/volumes/maddydata/_data/tls

证书需命名 fullchain.pem 和 privkey.pem。

第四步 获取DKIM值并添加解析 ​

当容器正常运行后，在 /var/lib/docker/volumes/maddydata/_data/dkim_keys 路径下会有个 lancet.icu_default.dns 的文件。

复制文件内容，在域名解析处增加解析

  • 主机记录值: default._domainkey
  • 记录类型： TXT
  • 记录值： 复制的内容

第五步 为域名添加其他解析 ​

提示

表格中 mail.example.com example.com 请换成自己的域名

记录类型域名值
Amail.example.com服务器ipv4地址
Aexample.com服务器ipv4地址
MXexample.commail.example.com
TXTmail.example.comv=spf1 mx ~all
TXTexample.comv=spf1 mx ~all
TXT_dmarc.example.comv=DMARC1; p=quarantine; ruf=mailto:postmaster@example.com
TXT_mta-sts.example.comv=STSv1; id=1
TXT_smtp._tls.example.comv=TLSRPTv1;rua=mailto:postmaster@example.com

第六步 使用 ​

注意

服务器需放开端口25、143、993、465、587

添加用户

在 1Panel 面板中 依次点击进入:

容器 -> 1Panel-maddy-mail-02ol -> 终端

连接终端后 输入命令 适当命令创建账户

# 创建登录账户
maddy creds create postmaster@example.com.cn

# 创建存储账户
maddy imap-acct create postmaster@example.com.cn

# 可以查看账户列表
maddy creds list
maddy imap-acct list

# 查看账户下的邮箱分类
maddy imap-mboxes list postmaster@example.com.cn

所有 example.com.cn 部分 需换成自己的域名

测试 使用其他邮箱向刚创建的邮箱地址发送邮件

使用以下命令查看收件箱

# 查看收件箱
maddy imap-msgs list postmaster@kafuuchino.com.cn INBOX

至此邮件服务器搭建完毕!

开源营销工具：Mautic ​

Mautic 是一款免费、开源、可定制的营销自动化软件，支持社交媒体营销、通讯管理、邮件营销、自动广告营销等功能。

Mautic 官网

第一步 下载Mautic ​

进入下载页面

填写问卷后，点击下载。

第二步 创建环境 ​

进入1panel面板>网站>运行环境，创建运行环境php，推荐 8.0.30 版本，建议重命名，区分其他php运行环境。

扩展处选择：mysqli、pdo_mysql、zip、gd、redis、memcache、mecached、imap、intl

进入1panel面板>数据库，创建数据库。

第三步 安装Mautic ​

创建网站，一键部署切换为 运行环境 ,PHP选择刚刚创建的php名称。

域名建议：mautic.example.com,然后部署证书。

点击网站目录，将下载的Mautic zip包上传至 /index 文件夹下，解压到此文件。完成安装，浏览器输入域名，如 mautic.example.com，填写创建的MySQL相关信息。

404

布署后mautic打开页面报错404，尝试使用 mautic.example.com/s/login 进行登录。

第四步 设置 ​

点击右上角的⚙︎按钮，进入Configuration，填写 Email Settings，设置成功后才能顺利发邮件。

进入Integrations>Plugins，如果有使用相关插件或服务，可以进行授权。

第五步 联系人和分组 ​

Contacts 与 Companies 分别对应C端联系人和B端联系人。

Segment是对联系人分组。

可以通过右上角的 import 导入，建议表格里的联系人数量在 1000~2000 左右。

第六步 EDM构建器和素材引用 ​

点击 Componets>Emails，新建一封 Email，选择 Segment Email，通过右上角的Builder就能进入到EDM邮件构建器。

可以将营销的素材上传至 Componets>Assets，然后复制链接，就能引用该素材。

后记 ​

在选择开源项目时，虽然可以获得免费的代码和社区支持，但适配现有系统或满足特定需求可能会变得相对复杂。开源软件通常需要一定的技术背景和定制化工作，这可能包括修改代码、解决兼容性问题等。可以考虑采用付费解决方案或许是一个更高效的选择。
